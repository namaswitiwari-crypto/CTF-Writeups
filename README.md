# PicoCTF Write-ups – Beginner & General Skills

## Overview
This repository contains my detailed write-ups for **PicoCTF challenges by [CyLab Academy](https://learn.cylabacademy.org/)**  
It focuses on **Beginner Skills** and **General Skills** problem sets.  
Each section documents the tasks, commands used, explanations, and key takeaways — making it a structured learning resource for cybersecurity fundamentals.

---

## Repository Structure
- **Beginner Skills**
  - Section 1 → Basic Linux commands & file handling
  - Section 2 → CyberChef encoding/decoding
  - Section 3 → Permissions, archives, website inspection, `strings`, `grep`, robots.txt
  - Section 4 → Python wrangling & password cracking (PW1–PW5)
  - Section 5 → SVG metadata, big archives, Java source analysis, reverse engineering, buffer overflow

- **General Skills**
  - Problem Set 1 → Introductory tasks
  - Problem Set 2 → Bash scripting & command-line
  - Problem Set 3 → Problem-solving with Linux utilities
  - Problem Set 4 → Advanced scripting & logic puzzles
  - Problem Set 5 → Final set with mixed challenges

---

## Skills Learned
- Linux basics: `chmod`, `cat`, `grep`, `strings`, `unzip`
- File permissions and recursive searching
- Encoding/decoding with CyberChef & Base64
- Website inspection (`robots.txt`, source code analysis)
- Python scripting, hashing (`hashlib`, SHA-256), brute-force & dictionary attacks
- Reverse engineering logic from source code
- Buffer overflow exploitation basics
