# General Skills - Problem Set 1

## Challenge: CyberChef Basics
### Task
Use CyberChef to decode or convert the data provided in the challenge.

### Steps I Took
1. Opened CyberChef in the browser.  
2. Dragged the required operation (e.g., **From Base64**, **ROT13**).  
3. Pasted the encoded string into the input box.  
4. Ran the recipe to get the decoded output.  

### Example
Input:

42 (base10)

Operation:
- Convert --> to base (radix = 2) 

Output:
101010


Another case:  
Input was in hexadecimal, so I used **From Hex** to convert it into ASCII text.

### Explanation
CyberChef is a powerful web tool that allows encoding, decoding, and data transformations through drag‑and‑drop “recipes.”  
Instead of writing scripts, I can quickly apply operations like **Base64 decode**, **Hex to ASCII**, or **ROT13**.  
This challenge helped me recognize common encodings and practice converting them efficiently.


## Takeaways
- Learned how to use CyberChef’s drag‑and‑drop recipes.  
- Practiced converting between different bases (decimal --> binary and hexadecimal --> decimal).  
- Understood how to decode Hex into ASCII text.  
- Realized CyberChef saves time compared to manual scripting.
